# warp-gate
Rapidly deploy modern web apps.

Includes:
- Webpack
- React 16.8
- Styleguide
- NURVUS UI Components

### Getting Started
1. `npm install` to install
2. `npm run start` to run app
3. `http://localhost:8080` to view in browser.

### Build
1. `npm run build` to generate a build to `/dist` directorry.
